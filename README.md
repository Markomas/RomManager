# RomManager

RomManager is a handheld client for the Romm App, designed to help you manage and download your ROM collection. 

## Features

- **Browse and Manage ROMs**: Connect to the Romm API to view and manage your game library.


## Technology Stack

- **Backend & Frontend**: Go
- **Graphics/UI**: [go-sdl2](https://github.com/veandco/go-sdl2)
- **Database**: SQLite with [GORM](https://gorm.io/)
- **Configuration**: YAML
